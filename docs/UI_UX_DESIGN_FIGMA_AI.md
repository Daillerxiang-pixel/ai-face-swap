## 1. 设计风格定义

### 设计理念

**关键词：** 简洁 · 活力 · 专业 · 易用

- **简洁** - 减少视觉干扰，突出核心功能
- **活力** - 使用渐变色彩，传达 AI 科技感
- **专业** - 遵循平台设计规范，保证体验一致性
- **易用** - 三步完成创作，降低使用门槛

### 设计原则 (UI/UX Pro Max)

根据 UI/UX Pro Max 技能的 **161 条推理规则**，本项目遵循以下核心原则：

| 优先级 | 类别 | 关键检查项 |
|--------|------|-----------|
| **CRITICAL** | 可访问性 | 对比度 4.5:1、触摸目标 44×44px、键盘导航 |
| **CRITICAL** | 触摸交互 | 最小点击区域、加载反馈、错误提示 |
| **HIGH** | 性能优化 | 图片懒加载、骨架屏、CLS < 0.1 |
| **HIGH** | 样式选择 | 匹配产品类型、一致性、SVG 图标 |
| **HIGH** | 响应式 | 移动优先、断点系统 (375/768/1024/1440) |

## 3. 字体系统

### 字体家族

```css
/* 中文 */
font-family: 'Noto Sans SC', 'PingFang SC', 'Microsoft YaHei', sans-serif;

/* 英文 */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* 数字 */
font-family: 'Inter', 'Roboto Mono', monospace;
```

### 字号规范

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| **超大标题** | 32px | Bold 700 | 1.2 | 页面大标题 |
| **大标题** | 24px | Bold 700 | 1.3 | 模块标题 |
| **标题** | 20px | SemiBold 600 | 1.4 | 卡片标题 |
| **副标题** | 18px | SemiBold 600 | 1.5 | 分组标题 |
| **正文大** | 16px | Regular 400 | 1.6 | 主要正文 |
| **正文** | 14px | Regular 400 | 1.5 | 常规正文 |
| **正文小** | 13px | Regular 400 | 1.5 | 辅助说明 |
| **提示文字** | 12px | Regular 400 | 1.4 | 提示、标签 |
| **极小文字** | 11px | Medium 500 | 1.4 | Tab 标签、徽章 |

### 字重使用

| 字重 | 值 | 使用场景 |
|------|-----|---------|
| Regular | 400 | 正文内容 |
| Medium | 500 | 强调文字、按钮 |
| SemiBold | 600 | 标题、Tab 选中 |
| Bold | 700 | 大标题、数字展示 |

```

   - 渐变色：#6366f1 → #ec4899
```
